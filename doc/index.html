shorter.h

A lightweight single-header sorting library for C. Sort integers, strings, floats, and custom data types using one simple generic API.

About

shorter.h provides a reusable generic sorting implementation for C using void * and compare functions.

It supports built-in compare helpers for common C data types and includes an array length macro.

Installation

project/
├── main.c
└── shorter.h
#include "shorter.h"
gcc main.c -o app

length()

Get Array Size

Returns the number of items in an array. 

length(array)
int nums[] = {1,2,3,4};

printf("%d", length(nums));

sort()

Generic Sorting

sort(array, length, sizeof(type), compareFunction);
Parameter Description
array Array to sort
length Total number of elements
sizeof(type) Size of element type
compareFunction Compare callback
